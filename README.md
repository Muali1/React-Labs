# REACT LAB4

## Features
- Shopping cart using Redux
- Cart page with add/remove products
- Arabic/English language support with RTL/LTR
- Product listing and search

## Technologies
- React 19.2.0
- Redux Toolkit
- React Router
- Tailwind CSS
- Ant Design
- Axios

## Routes
- `/` - Home page
- `/product/:id/:slug/` - Product details
- `/cart` - Cart page
